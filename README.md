Задание: Разработать консольное приложение для управления личной библиотекой с использованием объектно-ориентированного программирования.
Приложение должно демонстрировать практическое применение основных концепций ООП: классы, объекты, инкапсуляция, наследование и полиморфизм.

Функционал:
Система управления библиотекой - консольное приложение для учета книг и читателей. Позволяет добавлять книги, регистрировать пользователей, выдавать и принимать книги обратно. 
Для работы нужен C++17, CMake 3.10 и любой компилятор с поддержкой C++17. Чтобы собрать программу: создайте папку build, перейдите в нее, выполните cmake .. и затем make. 
Для запуска: ./library на Linux/Mac или library.exe на Windows. 
При запуске появляется меню: 1 - показать все книги, 2 - показать всех читателей, 3 - добавить новую книгу, 4 - зарегистрировать читателя, 5 - выдать книгу читателю, 6 - принять книгу обратно, 7 - найти книгу по ISBN, 8 - показать профиль читателя, 9 - сохранить данные в файл, 10 - выход. 
Пример использования: выберите 3, введите название, автора, год и ISBN книги; выберите 4, введите имя читателя; выберите 5, чтобы выдать книгу; при выходе через пункт 10 данные сохраняются автоматически. 
Программа проверяет корректность года издания (от 1450 до текущего года), ограничивает количество книг у читателя (не более 3), сохраняет все данные в файл data/library_data.txt и загружает их при следующем запуске. 
Файлы проекта лежат в папке src, сохраненные данные - в папке data, сборка - через CMakeLists.txt.

Техническая часть:
Класс Book - хранит данные книги (название, автор, год, ISBN, доступность). 
Класс User - хранит данные читателя (имя, ID, взятые книги). 
Класс Library - главный класс, управляет книгами и пользователями, содержит методы добавления/поиска/выдачи. 
Main.cpp - консольное меню с циклом и switch-case. 
Данные сохраняются в текстовый файл формата: блок BOOK с полями, разделитель ---USERS---, блок USER с полями. 
При запуске loadFromFile() читает файл, парсит строки, создает объекты. 
При сохранении saveToFile() записывает все объекты в файл.
